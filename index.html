Project 3 • To-Do Today

To-Do Today provides information about local events in select cities that are happening Today! Users can browse information about weather, concerts, festivals, sporting events, breweries and more.

Built with HTML + CSS + JavaScript + JSON

To-Do Today's functionaility comes from combining five APIs: OpenBreweryBD, OpenWeatherMap, PredictHQ, TicketMaster and Zomato.

Built by Three Guys and an Aussie

To-Do Today was a team effort built by Dylan Cooper, Matthew Everett, Justin Gardner and myself. The project was completed in a five day scrum sprint.
