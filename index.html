Project 1 • Taskwork

A project management tool allowing users to track project tasks, review those in flight and boost team efficiency. Built by developers for development teams.


Built with HTML + CSS + Node.js + Express + Postgres

Users create projects and track their overall status through sprints. Each project is linked to its own task page, where users can add tasks and advance them through different stages of completion.

Built by The Harmonies

Taskwork was a team effort built by Luke Brazil, Jai Gokhale, Kevin Jeffers and myself. The app was conceptualized and designed as part of our DigitalCrafts Coding Bootcamp to showcase Fullstack Development Skills.

Connect with the team on LinkedIn Luke, Jai, Kevin and Harmony!


Project Features.

User Login and Form Validation
The app allows users to create an account or sign in as a guest.

Create Projects and Tasks
Users have access to their unique projects and tasks, can progress them through different stages of completion and delete those no longer needed.



My Contributions.

Product Manager and Designer
As the Project Manager and Designer, it was my responsibility to design a mobile-first app that was fully responsive while keeping the project on schedule. I empowered our development team to focus on Taskwork's MVP, facilitated daily stand-ups and utilized Trello to track progress.



Project 2 • To-Do Today

To-Do Today provides information about local events happening today in select cities! Users can browse information about weather, concerts, festivals, sporting events, breweries and more.


Built with HTML + CSS + JavaScript + JSON

To-Do Today's functionality comes from combining five APIs: OpenBreweryBD, OpenWeatherMap, PredictHQ, TicketMaster and Zomato.

Built by Three Guys and an Aussie

To-Do Today was a team effort built by Dylan Cooper, Matthew Everett, Justin Gardner and myself. The project was completed in a five day scrum sprint.

Connect with the team on LinkedIn Dylan, Matthew, Justin and Harmony!


Project Features.

Provide Weather and Local Events
Users get access to the local weather and a list of events that are happening today.

Select Cities
To-Do Today is currently available in Atlanta, Houston, Seattle and Tampa.

Category Selections
Select categories to add or remove events of interest.



My Contributions.

Product Designer
As the Lead Product Designer, it was my responsibility to design a mobile-first app that was fully responsive and functional on Chrome, iOS and Andriod. Logo creation, graphic design and brand implementation all fell under my responsibility.



Project 3 • Sean vs AI

Sean vs AI is a fully interactive puzzle game built in Python using Pygame. Playing the game as Sean, a master coder who has a date with destiny, the goal is to solve coding problems that would be familiar to an entry-level developer. Tackle questions on dictionaries, booleans, if and then statements, and for loops to defeat the Evil AI and save DigitalCrafts.


Built with Python3 + Pygame

Sean vs AI's gameplay comes from combining text based questions and answers, with retro inspired graphics that utilize Pygame for player animation and screen bliting.

Built by Witty, Thought Provoking Team Name

Sean vs AI was a team effort built by Aliyah Gamble, Jai Gokhale, Eric Schorling and myself. The game was completed in a four day sprint only 2 weeks into our web development bootcamp.

Connect with the team on LinkedIn Aliyah, Jai, Eric and Harmony!


Project Features.

Multiple Levels and a Hidden Final Challenge
Playing as Sean, gamers will tackle four levels. Answering specific coding questions and keeping track of a randomized secret code, needed to proceed from level to level. The final level shows up once all the puzzles have been solved.



My Contributions.

Player Animation and Gameboard Designer
In creating Sean vs AI, my responsibilities included designing the multiple gameboards and animating the player. My design process included sketching to get initial ideas on paper, then tools such as Canva, Bitmoji, Pygame and Python3 to create the finished design.
